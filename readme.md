# Data Platform Cost Monitoring (FinOps Simulation)

Este projeto simula o monitoramento de custos de uma plataforma de dados, inspirado em ambientes como AWS e Databricks.

O objetivo é analisar o uso de recursos (processamento, armazenamento e queries) e gerar métricas que permitam identificar padrões de consumo, otimizar custos e detectar possíveis anomalias.

## Tecnologias
Python
Pandas
PySpark


## Funcionalidades
Simulação de dados de uso de plataforma
Cálculo de custos de processamento, armazenamento e queries
Geração de métricas de uso e custo
Identificação de possíveis anomalias


## Objetivo

Apoiar a tomada de decisão sobre uso eficiente de recursos em plataformas de dados, alinhado a práticas de FinOps e governança de dados.
